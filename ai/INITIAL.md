### Main objective
Assist me in creating an application for saving, categorising, and searching user notes. 

### Minimum functionalities: 

-The application should allow users to create accounts by providing a unique name.

- The application should record the date and time when content is added. 

- The application should analyse the entry semantically using the LLM model.  

- The application should generate keywords for each note, assign the entry to an existing category or create a new one. 

- The user can view saved notes, search for entries by date, category, phrases, keywords. The application should use the LLM model to find the searched content.
